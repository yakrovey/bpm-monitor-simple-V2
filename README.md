# Монитор задач BPM V2

Chrome-расширение (WebExtensions) для мониторинга задач **ПРЗ / ФРЗ / ПКМ** на дашборде IBM BPM.

## Браузеры

| Браузер | Как ставить |
|--------|-------------|
| **Chrome** | `chrome://extensions` → распакованное |
| **Edge** | `edge://extensions` → распакованное |
| **Яндекс.Браузер** | `browser://extensions` → распакованное |
| **Safari** | только через конвертер Xcode (`safari-web-extension-converter`) на macOS |

Если вы используете Firefox, то файлы для расширения можно скачать отсюда: https://github.com/yakrovey/bpm-monitor-simple-V2_mozilla

## Возможности

- Фоновый сбор статуса с дашборда «Работа»
- Вкладки ПРЗ / ФРЗ / ПКМ со счётчиками по цветам
- Рабочий таймер (пн–пт 09:00–18:00) от времени со страницы
- Уведомления по срокам и смена схем (Радио / ВОЛС) для ФРЗ и ПКМ
- Данные только локально в браузере

## Установка (Chromium)

1. Режим разработчика в расширениях  
2. «Загрузить распакованное» → эта папка  
3. Откройте BPM и нажмите «Проверить сейчас»  

Подробности — кнопка **«Инструкция»** в расширении (`help.html`).
