# Монитор задач BPM V2alt

Альтернативная Chromium-сборка: те же таймеры и схемы, что у V2, но данные снимаются с **трёх отдельных дашбордов**, а не с общего «Работа».

| Шаг | URL |
|-----|-----|
| ПРЗ | `https://workplace.ertelecom.ru/ProcessPortal/dashboards/SYSRP/4002` |
| ФРЗ | `https://workplace.ertelecom.ru/ProcessPortal/dashboards/SYSRP/4003` |
| ПКМ | `https://workplace.ertelecom.ru/ProcessPortal/dashboards/SYSRP/4004` |

## Как работает

1. При старте браузера расширение **ничего не делает**.
2. Кнопка **«Запустить»** открывает 3 закреплённые вкладки (4002 / 4003 / 4004), ждёт ~1 минуту и начинает опрос.
3. Дальше раз в минуту читает все три дашборда.
4. **«Проверить сейчас»** — сразу снимает данные; если вкладки закрыты — открывает их заново.
5. Можно выключить всплывающие уведомления кнопкой в popup (индикатор зелёный/красный).

Тип задачи задаёт дашборд-источник. Выбранные схемы ФРЗ/ПКМ сохраняются локально и восстанавливаются после нестабильного снимка.

Подробности — в `help.html` (кнопка «Инструкция» в расширении).

## Установка

1. `chrome://extensions` (или `edge://extensions` / `browser://extensions`)
2. Режим разработчика → **Загрузить распакованное**
3. Укажите папку `bpm-monitor-simple-V2alt`

Не ставьте обычную V2 и V2alt одновременно без нужды.

## Приватность

Только `chrome.storage.local` и сайт `workplace.ertelecom.ru`. Сторонних запросов нет.
