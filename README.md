# Мониторинг BPMS

Chrome-расширение (WebExtensions) для мониторинга задач **ПРЗ / ФРЗ / ПКМ / Монтаж** на дашборде IBM BPM (Дом.ru / ЭР-Телеком).

Текущая версия: **2.4.28**

## Браузеры

| Браузер | Как ставить |
|--------|-------------|
| **Chrome** | `chrome://extensions` → распакованное |
| **Edge** | `edge://extensions` → распакованное |
| **Яндекс.Браузер** | `browser://extensions` → распакованное |

## Возможности

- Фоновый сбор с дашборда `SYSRP/13202` (~раз в минуту), отдельное окно монитора
- Мягкое обновление таблицы без полной перезагрузки страницы
- Вкладки ПРЗ / ФРЗ / ПКМ (счётчики по цветам) и **Монтаж** (только количество: НКУ КРУС и ПКМ Подключение)
- Рабочий таймер (пн–пт 09:00–18:00) от даты со страницы
- Уведомления по срокам; схемы Радио / ВОЛС / Монтаж; автоподстановка схемы по СОС (медь / ВОЛС / P2P·P2MP·ДРОП) для ФРЗ/ПКМ
- Данные только локально в браузере

## Установка (Chromium)

1. Откройте `chrome://extensions/` (или аналог в Edge / Яндекс)
2. Включите **режим разработчика**
3. Скачайте файлы расширения в отдельную папку (в папке только файлы проекта)
4. **Загрузить распакованное расширение** → выберите эту папку
5. Закрепите иконку расширения, если она спрятана в меню расширений
6. Откройте BPM (дашборд 13202) и нажмите **Проверить сейчас**

Подробности — кнопка **«Инструкция»** в расширении (`help.html`).
