# Монитор задач BPM V2

Chrome-расширение (WebExtensions) для мониторинга задач **ПРЗ / ФРЗ / ПКМ** на дашборде IBM BPM.

## Браузеры

| Браузер | Как ставить |
|--------|-------------|
| **Chrome** | `chrome://extensions` → распакованное |
| **Edge** | `edge://extensions` → распакованное |
| **Яндекс.Браузер** | `browser://extensions` → распакованное |
| **Safari** | только через конвертер Xcode (`safari-web-extension-converter`) на macOS |

Если вы используете Firefox, то файлы для расширения можно скачать отсюда: https://github.com/yakrovey/bpm-monitor-simple-V2_mozilla

## Возможности

- Фоновый сбор статуса с дашборда «Работа»
- Вкладки ПРЗ / ФРЗ / ПКМ со счётчиками по цветам
- Рабочий таймер (пн–пт 09:00–18:00) от времени со страницы
- Уведомления по срокам и смена схем (Радио / ВОЛС) для ФРЗ и ПКМ
- Данные только локально в браузере

## Установка (Chromium/Chrome)

1. Режим разработчика в расширениях  chrome://extensions/
2. В правом верхнем углу есть режим разработчика, включите его.
3. Скачайте все файлы из списка выше в отдельно созданную папку на своем компьютере. в папке ничего кроме этих файлов быть не должно
4. в верхней левой части экрана барузера найдите кнопку «Загрузить распакованное расширение» → найдите там папку, куда вы скачали все файлы, нажмите кнопку открыть. Расширение само установится.
5. если вы не находите расширение, то скроее всего оно у вас спрятано. Найдите в верхнем левом углу значок расширений. обычно это иконка в виде кусочка пазла, справа от адресной строки. Нажмите на нее и закрепите расширение. 
6. Откройте BPM и нажмите «Проверить сейчас»  

Подробности — кнопка **«Инструкция»** в расширении (`help.html`).
Приятного пользования!
