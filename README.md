# Монитор задач BPM V2alt

Вариант расширения с теми же таймерами/уведомлениями, что V2, но данные берутся с **трёх отдельных дашбордов**:

| Шаг | URL |
|-----|-----|
| ПРЗ | `…/ProcessPortal/dashboards/SYSRP/4002` |
| ФРЗ | `…/ProcessPortal/dashboards/SYSRP/4003` |
| ПКМ | `…/ProcessPortal/dashboards/SYSRP/4004` |

## Вкладки

Расширение **само** открывает и держит три закреплённые фоновые вкладки и опрашивает их раз в минуту. Вручную держать их открытыми не обязательно — главное не закрывать и один раз авторизоваться в BPM.

Альтернатива «одна вкладка с переключением URL» намеренно не используется: медленнее и мешает, если смотрите дашборд.

## Установка

Как у Chromium-сборки V2: `chrome://extensions` → режим разработчика → загрузить распакованное → эта папка.

Не ставьте V2 и V2alt одновременно без нужды — оба мониторят BPM.
