Монитор задач BPM V2alt

Альтернативная сборка: задачи ПРЗ / ФРЗ / ПКМ читаются с трёх отдельных дашбордов (4002 / 4003 / 4004), а не с общего «Работа». Таймеры и схемы — как в V2.

1. Установка

Расширение для Chrome, Edge и Яндекс.Браузера. Загружайте папку bpm-monitor-simple-V2alt (не путать с обычной V2 и с Firefox-сборкой).

Google Chrome

  1. Откройте chrome://extensions
  2. Включите Режим разработчика
  3. Загрузить распакованное расширение → папка bpm-monitor-simple-V2alt
  4. Разрешите уведомления, если спросит

Microsoft Edge

  1. Откройте edge://extensions
  2. Включите Режим разработчика
  3. Загрузить распакованное → та же папка V2alt

Яндекс.Браузер

  1. Откройте browser://extensions или Меню → Дополнения → Расширения
  2. Включите режим разработчика
  3. Загрузите распакованное расширение из папки bpm-monitor-simple-V2alt

После обновления файлов нажимайте «Обновить» на карточке расширения.

Не включайте одновременно обычную V2 и V2alt без нужды — оба будут мониторить BPM.

2. Источники данных

В отличие от V2 (один дашборд «Работа»), V2alt держит три закреплённые вкладки и раз в минуту читает каждую:

ШагДашборд
ПРЗ https://workplace.ertelecom.ru/ProcessPortal/dashboards/SYSRP/4002
ФРЗ https://workplace.ertelecom.ru/ProcessPortal/dashboards/SYSRP/4003
ПКМ https://workplace.ertelecom.ru/ProcessPortal/dashboards/SYSRP/4004

Тип задачи задаёт сам дашборд (4002 → ПРЗ, 4003 → ФРЗ, 4004 → ПКМ). Достаточно один раз войти в BPM в любой вкладке — сессия общая.

3. Первый запуск

  1. При старте браузера расширение ничего не делает (режим ожидания).
  2. Откройте иконку расширения → нажмите Запустить.
  3. Откроются 3 закреплённые вкладки дашбордов (4002 / 4003 / 4004).
  4. Если появится окно входа — авторизуйтесь в BPM.
  5. Через ~1 минуту будет первая проверка. Дальше опрос раз в минуту.

Вкладки лучше не закрывать. Если закрыли — нажмите Проверить сейчас: закрытые откроются снова, открытые просто перечитаются.

4. Кнопки в окне расширения

  • Запустить — первый старт: открыть 3 дашборда и включить мониторинг.
  • Проверить сейчас — сразу снять данные; при необходимости переоткрыть закрытые вкладки 4002/4003/4004.
  • не показывать уведомления / показывать уведомления — выключить или включить всплывающие уведомления. Рядом индикатор: ● зелёный — уведомления вкл, ● красный — выкл.
  • Инструкция — эта страница.

Даже при выключенных всплывающих уведомлениях сбор задач и таймеры в окне расширения продолжают работать. Отдельно: пороги по срокам приходят только в рабочее время (пн–пт 09:00–18:00).

5. Вкладки ПРЗ / ФРЗ / ПКМ

  • В окне расширения три вкладки: ПРЗ, ФРЗ, ПКМ
  • В каждой — только задачи этого шага
  • На кнопках вкладок цветные счётчики:
    зелёные жёлтые красные синие (просрочка) белый — ВОЛС (если есть)

Если снимок ФРЗ на мгновение пустой (нестабильная отрисовка дашборда), список не обнуляется сразу — выбранные схемы запоминаются и восстанавливаются.

6. Таймер и рабочее время

  • Отсчёт от даты/времени со страницы (как в мастер-системе)
  • Считается только пн–пт, 09:00–18:00; ночью и в выходные — пауза
  • Сбор статуса — всегда (когда монитор запущен); уведомления по срокам — только в рабочее время
  • В уведомлениях есть клиент и адрес / название

7. Смена схемы на ФРЗ и ПКМ

На шагах ФРЗ и ПКМ можно сменить схему уведомлений и сроков. На ПРЗ схема одна.

  1. Откройте вкладку ФРЗ или ПКМ
  2. Нажмите на нужную задачу в списке
  3. Выберите:
    • Обычная схема — стандартные сроки
    • Радио — другие пороги и тексты
    • Схема 2 ВОЛС — таймер выкл., заявка белая; для ФРЗ — напоминание раз в 48 ч, для ПКМ — без уведомлений
  4. Нажмите Применить

Выбор схемы сохраняется локально и не должен сбрасываться при кратковременном «мигании» счётчика ФРЗ.

9. Если что-то не работает

  • Нет задач / статус «Пауза» — проверьте вход в BPM на любой из вкладок 4002/4003/4004, затем Проверить сейчас.
  • Сообщение «ПОХОЖЕ, ВХОД В BPMS НЕ ВЫПОЛНЕН» — авторизуйтесь; после успешного входа предупреждение само прекращается.
  • Закрыли вкладки дашбордов — снова Проверить сейчас (или перезапустите через «Запустить», если монитор был остановлен).
  • После правок файлов — «Обновить» на chrome://extensions.

10. Приватность

  • Данные только в браузере (chrome.storage.local)
  • Сеть — только workplace.ertelecom.ru
  • На сторонние серверы ничего не уходит
