Мониторинг BPMS

Инструкция к текущей сборке: вкладки ПРЗ / ФРЗ / ПКМ / Монтаж, таймер, схемы, уведомления и фоновый сбор.

версия 2.4.28

1. Установка

Расширение для Chrome, Edge и Яндекс.Браузера.

Google Chrome

  1. Откройте chrome://extensions
  2. Включите Режим разработчика
  3. Загрузить распакованное расширение → папка проекта
  4. Разрешите уведомления, если браузер спросит

Microsoft Edge

  1. edge://extensions → режим разработчика → загрузить распакованное

Яндекс.Браузер

  1. browser://extensions → режим разработчика → загрузить распакованное

После обновления файлов нажмите «Обновить» на карточке расширения.

2. Первый запуск и фоновый монитор

  1. Войдите в BPM и откройте дашборд:
    https://workplace.ertelecom.ru/ProcessPortal/dashboards/SYSRP/13202
  2. Иконка расширения → Проверить сейчас
  3. Расширение само держит отдельное окно/вкладку монитора с дашбордом 13202. Его не нужно держать на переднем плане: список обновляется в фоне.
  4. Полную перезагрузку страницы расширение не использует — мягкое обновление таблицы (поиск / refresh на странице), чтобы не мешать работе.

Не закрывайте окно монитора и не выходите из BPM. Если список пуст — откройте дашборд 13202 и снова нажмите «Проверить сейчас».

3. Что отслеживается

  • ПРЗ, ФРЗ, ПКМ — по теме задачи
  • Монтаж — «ПКМ: состояние НКУ КРУС» и «ПКМ: Подключение» (отдельная вкладка)
  • Темы вроде «Шаг 1.2 / 3.1 / 5.1» и «отложен» не берутся

4. Окно расширения

  • Заголовок: Мониторинг BPMS
  • Статус справа: активен / загрузка / ошибка
  • Краткий блок: последняя проверка, число активных, период фона, рабочие часы
  • Кнопки: Проверить сейчас (синяя), Открыть BPM, Инструкция
  • Вкладки шагов:
    • ПРЗ / ФРЗ / ПКМ — цветные счётчики (зелёный / жёлтый / красный / синий; белый — ВОЛС)
    • Монтаж — только общее число задач, без цветовых пилюль
зелёные жёлтые красные синие (просрочка) белый — ВОЛС / монтаж

5. Как работает сбор и таймер

  • Фоновая проверка списка — примерно раз в 1 минуту
  • Таймер — от даты/времени со страницы (колонка вроде «Дата создания экземпляра»)
  • Считается только пн–пт, 09:00–18:00; ночью и в выходные таймер на паузе
  • Сбор статуса — всегда; уведомления — только в рабочее время
  • В уведомлении есть клиент и адрес / название
  • Для ФРЗ/ПКМ схема может подставиться автоматически по колонке СОС (медь → обычная, ВОЛС, P2P/P2MP/ДРОП → радио); вручную схему всё равно можно сменить кликом по задаче

6. Смена схемы (ФРЗ и ПКМ)

На ПРЗ схема одна — менять нечего. На Монтаж схема фиксирована.

  1. Откройте вкладку ФРЗ или ПКМ
  2. Кликните по задаче в списке
  3. Выберите схему и нажмите Применить:
    • Обычная — стандартные сроки
    • Радио — другие пороги и тексты (СОС: P2P, P2MP, ДРОП)
    • Схема 2 ВОЛС — таймер выкл., карточка белая; ФРЗ — напоминание раз в 48 ч, ПКМ — без уведомлений
    • Монтаж (только для ПКМ) — режим монтажа / НКУ КРУС / Подключение

Текущая схема видна в карточке задачи.

7. Вкладка «Монтаж»

  • Сюда попадают заявки ПКМ: состояние НКУ КРУС и ПКМ: Подключение
  • Таймер и цветные зоны выключены
  • На кнопке вкладки показывается только количество
