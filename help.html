Монитор задач BPM V2

Инструкция: вкладки ПРЗ/ФРЗ/ПКМ, сроки, цвета, уведомления и смена схем.

1. Установка в разных браузерах

Это расширение для Chrome, Edge и Яндекс.Браузера. Если вы используете Firefox, то файлы для расширения можно скачать отсюда: https://github.com/yakrovey/bpm-monitor-simple-V2_mozilla

Google Chrome

  1. Откройте chrome://extensions
  2. Включите Режим разработчика
  3. Загрузить распакованное расширение → папка bpm-monitor-simple-V2
  4. Разрешите уведомления, если спросит

Microsoft Edge

  1. Откройте edge://extensions
  2. Включите Режим разработчика
  3. Загрузить распакованное → та же папка

Яндекс.Браузер

  1. Откройте browser://extensions или Меню → Дополнения → Расширения
  2. Включите режим разработчика
  3. Загрузите распакованное расширение из папки проекта

Chrome / Edge / Яндекс основаны на Chromium — используется файл manifest.json как есть.

Apple Safari (macOS)

  1. Safari не загружает папку Chrome «как есть» — нужен Safari Web Extension
  2. На Mac с Xcode: 
    xcrun safari-web-extension-converter "/путь/к/bpm-monitor-simple-V2" --app-name "BPM Monitor V2"
  3. Откройте созданный проект в Xcode → Run
  4. Safari → Настройки → Расширения → включите «BPM Monitor V2»
  5. Разрешите доступ к сайту workplace.ertelecom.ru

Для распространения вне вашего Mac нужна учётная запись Apple Developer. На Windows/Linux Safari-расширение не ставится.

После любых правок файлов в Chromium-браузерах нажимайте «Обновить» на карточке расширения.

2. Первый запуск

  1. Откройте дашборд Работа и войдите:
    https://workplace.ertelecom.ru/ProcessPortal/dashboards/SYSRP/RESPONSIVE_WORK
  2. Вкладку можно свернуть, но не закрывайте — BPM сам подставляет задачи, расширение страницу не перезагружает
  3. Иконка расширения → Проверить сейчас

3. Что отслеживается

  • Только темы с ПРЗ, ФРЗ или ПКМ
  • «Шаг 1.2 / 3.1 / 5.1» и прочие шаги игнорируются

4. Вкладки ПРЗ / ФРЗ / ПКМ

  • В окне расширения три вкладки: ПРЗ, ФРЗ, ПКМ
  • В каждой вкладке — только задачи этого шага
  • На кнопках вкладок цветные счётчики:
    зелёные жёлтые красные синие (просрочка) белый — ВОЛС (если есть)
  • Число на счётчике = сколько заявок сейчас в этом цвете на шаге

5. Таймер и рабочее время

  • Отсчёт от даты/времени со страницы (как в мастер-системе)
  • Считается только пн–пт, 09:00–18:00; ночью и в выходные — пауза
  • Сбор статуса — всегда; уведомления — только в рабочее время
  • В уведомлениях всегда есть клиент и адрес / название

6. Смена схемы на ФРЗ и ПКМ

На шагах ФРЗ и ПКМ можно сменить схему уведомлений и сроков. На ПРЗ схема одна — менять нечего.

  1. Откройте вкладку ФРЗ или ПКМ
  2. Нажмите на нужную задачу в списке
  3. В окне «Сменить схему в уведомлении?» выберите:
    • Обычная схема — стандартные сроки
    • Радио — другие пороги и тексты уведомлений
    • Схема 2 ВОЛС — таймер выключается, заявка белая; для ФРЗ — напоминание раз в 48 ч, для ПКМ — без уведомлений
  4. Нажмите Применить

Текущая схема у задачи видна в карточке («Схема: Обычная / Радио / ВОЛС»).

8. Кнопки

  • Проверить сейчас — считать задачи с дашборда
  • Открыть / показать вкладку BPM — перейти к «Работа»
  • Инструкция — эта страница

Если список пуст: откройте дашборд «Работа», обновите расширение и снова нажмите «Проверить сейчас».

9. Приватность

  • Данные только в браузере (chrome.storage.local)
  • Сеть — только workplace.ertelecom.ru
  • На сторонние серверы ничего не уходит
